## 须知：请勿传播731式笑料（以伤害生命为代价的笑料）。娱乐请务必分辨清楚玩笑到底能不能开，不是出于爱的笑，可能仅是无负罪感的恶。当731式笑料真实发生时就是人间惨剧，传播的人也形同杀人的帮凶。
#### 阅读说明：喜乐是好事，但请务必分清场合、摆正身份。分清场合是指，在私人场合（家、聚会等）可以随兴，而在公众场合就不该如此，应当明礼、收敛，否则喧哗会影响别人。摆正身份是指，年纪尚小可以不用顾虑多少，所谓童言无忌；可身为长辈，肩膀上是有份量的，应当先看到言语背后的事情是否严肃，审视下是不是已经存在或将产生恶劣问题。屠杀的刽子手杀人取乐时也同样笑得没遮没拦。
#### 一些其他说明：本页文字的目的在于明快有趣地阐明些道理，需要考虑到普通受众的接受程度，所以风格较为轻松诙谐，不会像学术理论一样只为追求真实严谨。如果读者您觉得反面案例有所冒犯，均纯属巧合。

## 谎言背后（Funny Phrase 3）
###### 拒绝异性总说，“你是个好人”，实际上是社会不公、好人太少，好人很难变得富有，TA不想喝西北风；保持暧昧总说，“只是我妹妹（情感上的确是妹妹，但只要不是血亲就不影响繁衍）”、“我去做头发（这其实是一则多义词案例，断句以后可以更准确地表达意思，做头、发。做头在中文俚语中又指可做的东西、活计，比如今天的生意很有做头；发在读fā时，有花开放的意思，文有记载‘野芳发而幽香’，类似意思可以被引用为盛开。完整的意思连起来是‘我去做些像花一样盛放的感兴趣事情’）”，实际上两性对立、相互使绊子，性自由诉求得不到认可，说实话会被打；等等。但我在想这是些什么鬼逻辑，是无法解决的社会问题搪塞和扭曲了人们的语言。如果社会问题得到解决，善良才是社会的主旋律，言论自由（造成恶劣影响还是要负责的）。那么拒绝异性时会说“你是个坏人”，想要暧昧时会说“我不想谈恋爱，TA是我的性伴侣”。当我们仇视一个个常人的从众谎言时，更该考虑的是背后的社会陈疴，而不该去恨无助才随波逐流的百姓。所以这虽然是一个笑话，但是也反应了我们的方向。成功了自然会有鲜花和掌声，失败了曝尸半道也算是殊荣。这些谎言只是在静静地诉说一个事实，所有问题最终只是解决社会问题还是被社会问题解决的问题。

##### >>>> 语言在一般情况下只是被用作了工具，真假会给目的让道；只有热爱语言，才会将其视为生命，一丝一毫的虚伪都是堕落。举例来说，爱慕学姐的学弟以借书学习为由邀约，这里借书到底是不是为了学习并不重要，重要的是借书可以带来一次交往的机会。用绝对的真话来要求普通人，属于刁难。对谎言不应过度严苛，否则只容许真话也会造成业障。讲真话难，强行要求人完成做不到的事只会催生更多虚伪。对待平常生活中的交流应该以后果为考量重心，没有酿成恶劣影响就应该尽可能给予宽容。但是，不鼓励人说谎，通常比谎言更好的选择有很多，如行动、沉默、请人帮忙等。
##### >>>> 谎言对人存在负面作用，生效后会造成误导影响，还会断送彼此间信任。谎言的后果在场合稍正式一些时将变得很严重，残酷的真相也依然比诡魅的谎言要好得多。讲个直观的例子，把硫酸说成柠檬水。再举个例子，欺骗别人说高压电很好玩、很安全，那性质就是谋杀，只是在科普工作落实卓越的情况下通常谋杀未遂，但如果听众里有懵懂无知的孩童呢？轻信谎言的人会在人生的终点痛恨欺骗者，因为谎言的误导使得他人做出错误的人生决断，看似没有什么却实际毁掉了一整个人生。刺耳的真话至少可以让人在残酷的事实面前清醒，可以在现实条件下做出最优的人生选择，人生不会留有遗憾。谎言并不会改变悲惨的人生，但是真相可以让人争取一个改变命运的机会。而且，不说谎也是保护自身。谎言当时被拆穿并非最可怕，就怕真的有人信了，只要影响大到存在必要时，别人会不论真假地帮助说话者兑现承诺。
##### >>>> 谎言的负面作用还存在于事物本身，量变到一定程度以后，就会对其产生不可磨灭的伤害。因为谎言的本质是对语言的不当使用。如同拿诺基亚手机开核桃，并不是手机本就能这样用，只是手机质量好，但显然多少还是对手机造成了伤害。举例，鬼神之学从一开始的星际探索启蒙，到后面的招摇撞骗，现如今几乎没有人会相信这门学问。甚至，语言也不例外，谎言的泛滥会限制该语言的发展。因为所有的谎言最终并不是不了了之，而是由存在于文明中的其他真理承兑责任，需要不断有人处理谎言的灾害，但这并非没有限度。事态若是严重到了失控的地步，一个文明充斥满了谎言，其他文明就会不再相信这个文明的语言，久而久之该文明就会逐渐走向没落。
##### >>>> 其次我们再来分析事情，一般生活场景下口头说的话比起对人实实在在的伤害是次要因素，做一个善良的哑巴也比拿真理为幌子肆意伤害他人要好得多。只是讲真话难，还要达到目标更难，一般人做不到，在权衡了谎言和实话的后果之后，可能是说话者在两权相害取其轻后的选择。虽然“你是个好人”常常是一句谎言，但是这里的圆场作用远大于欺骗作用，背后的实话不加思索地说出来往往伤人，所以这句话更主要的属性是一句比较通用的圆场话。被分手了的男孩已经足够可怜，这时男孩到底是不是一个标准意义上的好人并不重要，比好人的真假更重要的是心灵上的安慰。讲类似“你是个loser”这种话丝毫不通人情，虽是实话但更是瞎话。听到这句圆场话一定是因为日积月累的结果，如果不反思一下自身是不是真的好，甚至就不去做好人了，那其实是再一次地带给别人伤害，更重要是对不起自己的父母。但其实组织和修饰下语言后讲真话更好，如“能有一段相守的缘分已很知足，但我等不起你，你好好努力加油，不要让这段感情在将来变得廉价而被人耻笑”、“非XXX不娶（嫁），请自重！”等。
##### >>>> 实话和真话并不是同一件事。不顾人的实话会伤人，伤人的话更是欠考虑的瞎话。很多时候的实话只是局部的真，说话者只是站在了自己的角度看待问题，站在交谈对象的角度往往是无足轻重的那一部分，可能心里想的是说话人这龟孙懂个什么东西在这里讲话不过脑地瞎bb。就像在别人的庆功宴上挑刺，这时挑刺的问题真假并不重要，重要的是取得里程碑式的胜利，需要鼓舞全员士气。不识趣的人挑的那些刺可能别人早就心知肚明，宴会结束后内部的自我检讨要比这严苛得多，一些挑刺的小问题是真，全员连年劳顿、士气需要激励更是真，两者相较当然选择更重要的那部分真。在很多不重要的场合下，对话语境中的事情存在许多真实的要素，不能只见树木不见森林，不涉及切身利益又照顾人感受的圆话要好过盲目地瞎说大实话。所以，如果不能讲话讲到不伤害他人又不坏事情，那么讲两句圆场话无可厚非，再不济就别说话了。但大是大非绝对不能如此，原则问题容不得半点含糊。那么什么是真话？考虑到了方方面面因素又反映客观事实的话是真话，否则说的话都存在缺漏考量点的失真，只是说话者角度的实话。正经地评价讲真话这事，人无法讲出绝对的真话，很可能主观想说真话但却客观说了失实的话。讲令人愉悦的真话还能达到目的是一件极难的事情，全球的外交家也没多少。
##### >>>> 用于简易区分判断欠考虑的实话和真话的一个问题，既然是真话又怎么会引起虚假的反应？多是因为没有全面看待问题，片面局限的实话损害了观点以外立场的利益。这引起了不同立场群体下意识地祭出敷衍的借口，幌子是为了避免缺失考量点的实话将要造成的损失，借此保护己方利益。
##### 总结建议：瞎话千万别说，谎话尽量不说，实话想好再说，真话能说就说。

## "狗"字贬义组词的一些解释（Funny Phrase 7）
###### 首先要说的是，脏话能不用就不要用，都有替代的文字可以表达同样的意思，而且说多了无论怎样都会显得自身素质低下。只有真的是天怒人怨了，确确实实是脏话描述的情形了，这样子使用时才有效果，否则不实的描述毫无杀伤力，还只会显得自身粗鄙。

1. 放狗屁：狗几乎所有的言语都是围绕效忠主人展开，所以不管什么话都向着主人，说多了就好像放屁一样。不过这没有什么问题，世界既需要真知灼见，也需要情感联结，但千万别狗屁不通。
2. 狗东西：形容一个人像狗的性格说明很忠诚，是褒义的。但形容狗东西说明只是看上去像狗的东西，是条假狗，只是打着忠诚的幌子而贪图利益，既不是人连动物都不是。逻辑很简单，奸侫之人喜欢伪装成忠义的样子，才好借此迷惑他人和牟取非分利益。与此意思相近的还有狗官、瞎了狗眼。对于狗，我们人不应该去计较，否则是在为难狗，而且对自身存在侮辱性极强的暗喻（活得还不如一条狗会做人）。所以人完全照着狗一模一样学就是彻彻底底的贬义，小事人可以忠于情感而不顾其他，但大是大非人应该忠于正义与真理。没有正义，人类将变得茹毛饮血，不如野兽。真正忠诚的人在面对已经确凿的真理时，应该警醒人明辨是非、劝诫人迷途知返，而绝非是装作狗一样的东西让人在错误的道路上越走越远。
3. 狗日的：因为狗太忠诚了，日完就回主人身边去了。通常词语用于两种情形，一种是形容做人有问题，辜负了别人的信任，伤害了别人真心，只配被利用；另一种是形容识人方面很不长进，很容易轻信胡话。
4. 日了狗：再怎么日都没啥感情，狗还是忠于主人，形容很倒霉没啥收获。通常泛指没做好信息准备工作而盲目付出，就比方说挖墙脚去挖别人家的承重柱一样。
